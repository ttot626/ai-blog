# AI Blog

8 周实战项目：基于 Spring Boot 的 AI 博客系统。

## 技术栈

- Java 21
- Spring Boot 3.4
- MySQL
- Lombok

## 当前进度

- [x] Day 1：Spring Boot 项目启动，`GET /hello` 接口

## 快速开始

```bash
mvn spring-boot:run
```

浏览器访问：http://localhost:8080/hello

预期返回：

```text
hello ai blog
```

## 项目结构

```text
src/main/java/com/example/xiangmu1/
├── Xiangmu1Application.java   # 启动类
└── controller/
    └── HelloController.java   # Web 接口
```
